MCP server: tools, resources & prompts
pe-orgair-server — 6 tools, 2 resources, 2 prompts (mcp/server.py)
MCP tools — @mcp_server.list_tools()
calculate_org_air_score
CS3 cs3_client.get_assessment()
Calculate Org-AI-R score. Returns: org_air, vr_score, hr_score, synergy_score, confidence_interval, dimension_scores. Input: company_id (required).
get_company_evidence
CS2 cs2_client.get_evidence()
Retrieve AI-readiness evidence. Filters by dimension (7 values + "all"). Returns: source_type, content[:500], confidence. Inputs: company_id, dimension, limit.
generate_justification
CS4 cs4_client.generate_justification()
Generate evidence-backed justification via RAG. Returns: score, level, level_name, evidence_strength, rubric_criteria, supporting_evidence[:5], gaps_identified.
project_ebitda_impact
CS3 ebitda_calculator.project()
Project EBITDA impact using v2.0 model. Returns: delta_air, conservative/base/optimistic scenarios, risk_adjusted, requires_approval flag.
run_gap_analysis
CS3+CS4 gap_analyzer.analyze()
Analyze gaps and generate 100-day plan. Calls cs3_client.get_assessment() then gap_analyzer. Returns: gap by dimension, priorities, initiatives, investment.
get_portfolio_summary
CS1 portfolio_data_service.get_portfolio_view()
Get fund portfolio summary. Returns: fund_id, fund_air (avg), company_count, companies [{ticker, org_air, sector}].
